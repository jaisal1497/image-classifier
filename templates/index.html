Multi-class Prediction Model

Using webscraping we downloaded ~3k images of Pikachu, Kanye and Cats. The model uses transfer learning on imagenet VGG-19 weights to train a multi-class Convolutional Neural Network(CNN) image classifier.The model accepts an arbitrary image as an input and it calculates three probabilities, one for each class (Kanye, Pikachu or cat). Our results predict the category with the highest probability.
